# Practice 4 - Generators, Dates, Math, JSON

Python standard library modules.

## Topics Covered

- `generators.py` — Generator functions with `yield`
- `dates.py` — Working with `datetime` and `timedelta`
- `math.py` — Math operations using the `math` module
- `json.py` — Parsing and working with JSON data

## Running

```bash
python generators.py
python dates.py
python math.py
python json.py
```
