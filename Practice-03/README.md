# Practice 3 - Functions, Lambda, Classes, Inheritance

Object-oriented programming and functional tools in Python.

## Topics Covered

### Functions (`functions/`)
- `basic_functions.py` — Defining and calling functions
- `function_arguments.py` — Positional, keyword, default arguments
- `return_values.py` — Returning single and multiple values
- `args_kwargs.py` — `*args` and `**kwargs`

### Lambda (`lambda/`)
- `lambda_basics.py` — Inline anonymous functions
- `lambda_with_map.py` — Using lambda with `map()`
- `lambda_with_filter.py` — Using lambda with `filter()`
- `lambda_with_sorted.py` — Using lambda with `sorted()`

### Classes (`classes/`)
- `class_definition.py` — Creating a class and objects
- `init_method.py` — Constructor with `__init__`
- `class_methods.py` — Instance methods
- `class_variables.py` — Shared class-level attributes

### Inheritance (`inheritance/`)
- `inheritance_basics.py` — Inheriting from a parent class
- `super_function.py` — Calling parent constructor with `super()`
- `method_overriding.py` — Overriding parent methods
- `multiple_inheritance.py` — Inheriting from multiple classes
