# Practice 10 - Pygame Games

Three extended Pygame game projects.

## Games

### Racer (`racer/`)
A top-down car racing game. Avoid enemy cars and collect coins.
- Arrow keys to move
- Coins appear randomly on the road
- Coin counter shown top-right
- Difficulty increases over time

### Snake (`snake/`)
Classic snake game with levels and scoring.
- Arrow keys to move
- Wall and self-collision detection
- Food spawns in random free cells
- Speed increases each level (every 3 foods eaten)

### Paint (`paint/`)
A simple drawing application.
- Tools: Pen, Line, Rectangle, Circle, Eraser
- 12-color palette
- Keyboard shortcuts: P / L / R / C / E
- DEL or N to clear canvas

## Installation

```bash
pip install pygame
```

## Running

```bash
python racer/main.py
python snake/main.py
python paint/main.py
```
