# Music Player

A simple Pygame music player that reads `.wav` / `.mp3` files from the `music/` directory.

## Controls

| Key | Action |
|-----|--------|
| P | Play / Resume |
| S | Stop |
| N | Next track |
| B | Previous track |
| Q | Quit |

## Running

```bash
python main.py
```

## Files

- `main.py` — Pygame UI and event loop
- `player.py` — `MusicPlayer` class
- `music/` — Place your audio files here
