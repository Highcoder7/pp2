# Moving Ball

A Pygame demo where you move a red ball around the screen using arrow keys. The ball stays within the window boundaries.

## Controls

| Key | Action |
|-----|--------|
| ↑ | Move up |
| ↓ | Move down |
| ← | Move left |
| → | Move right |

## Running

```bash
python main.py
```

## Files

- `main.py` — Pygame window and event loop
- `ball.py` — `Ball` class with movement and boundary logic
