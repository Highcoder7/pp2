# Mickey's Clock

An analog clock drawn with Pygame. Uses a Mickey Mouse hand image for clock hands (falls back to drawn lines if the image is missing).

## Running

```bash
python main.py
```

## Files

- `main.py` — Entry point, Pygame window setup
- `clock.py` — `MickeyClock` class with drawing logic
- `images/mickey_hand.png` — Hand image (optional)
