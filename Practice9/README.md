# Practice 9 - Game Development with Pygame

Three mini Pygame applications.

## Projects

- **Mickey's Clock** (`mickeys_clock/`) — Analog clock using Pygame rotation and image rendering
- **Music Player** (`music_player/`) — Audio player with keyboard controls
- **Moving Ball** (`moving_ball/`) — Ball that bounces within screen bounds

## Installation

```bash
pip install -r requirements.txt
```

## Running

```bash
python mickeys_clock/main.py
python music_player/main.py
python moving_ball/main.py
```
